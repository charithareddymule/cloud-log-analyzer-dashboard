# Cloud Log Analyzer Dashboard

This project analyzes server logs and visualizes insights using Python and Streamlit.

## Features
- Log classification (INFO, WARNING, ERROR)
- Interactive dashboard
- Data visualization of log events

## Technologies Used
- Python
- Pandas
- Streamlit

## Project Structure
cloud-log-analyzer
│
├── dashboards
│ └── app.py
│
├── logs
│ └── server_logs.txt
│
└── scripts

## How to Run

1. Install dependencies

pip install pandas streamlit

2. Run the dashboard

streamlit run dashboards/app.py
